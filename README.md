# WP Theme Lab

A foundational WordPress Block Theme built for learning **Full Site Editing (FSE)** and **theme.json** architecture.

## 🚀 Learning Goals
- Understand the minimum requirements for a Block Theme (`style.css`, `theme.json`, `templates/index.html`).
- Master the `theme.json` schema (Settings vs. Styles).
- Learn how to build reusable **Template Parts** for Headers and Footers.
- Explore the relationship between theme styles and core blocks.

## 🛠 Project Structure
- `/parts`: Reusable global components (Header, Footer).
- `/templates`: Page layouts.
- `theme.json`: The "brain" of the theme (Global Styles).

## How to use
1. Drop this folder into `/wp-content/themes/`.
2. Activate "Block Theme Lab" in the WordPress Dashboard.
